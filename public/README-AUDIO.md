# Audio Files Directory

Place your audio files here for the Christmas greeting website to work properly.

## Required Files

1. **christmas-music.mp3**
   - Background Christmas music
   - Should be 2-5 minutes long
   - Will loop automatically

2. **bell-sound.mp3**
   - Bell sound effect
   - 1-3 seconds
   - Triggered by bell button

3. **chime-sound.mp3**
   - Chime sound effect
   - 1-3 seconds
   - Triggered by chime button

## Where to Find Free Audio

- Pixabay: https://pixabay.com/music/
- Freesound: https://freesound.org/
- YouTube Audio Library: https://youtube.com/audiolibrary

See the main `AUDIO-FILES-NEEDED.md` file in the root directory for detailed instructions.
